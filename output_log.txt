[Infix Evaluation via Shunting-Yard]
  Infix:   ( 15000 + 5000 ) * 2
  Postfix: 15000 5000 + 2 *
  Result:  40,000.00 EGP

  Infix:   500000 - 120000 * 0.14
  Postfix: 500000 120000 0.14 * -
  Result:  483,200.00 EGP

  Infix:   ( 80000 + 40000 ) / ( 2 + 2 )
  Postfix: 80000 40000 + 2 2 + /
  Result:  30,000.00 EGP
